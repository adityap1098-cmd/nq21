# NQ21 PERFORMANCE

Back-office tracking app untuk bengkel motor NQ21. Catat transaksi internal, hitung komisi mekanik mingguan, visibility cash flow.

## Stack

React 18 + Vite + TypeScript + Tailwind v3 + shadcn/ui + Zustand
